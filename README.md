# Python - Programação com Threads

## 📖 Objetivo

Desenvolver exercícios acadêmicos utilizando threads para compreender conceitos de programação concorrente e execução simultânea de tarefas.

## 📂 Conteúdo

O repositório contém exercícios sobre:

- Threads;
- Execução concorrente;
- Compartilhamento de recursos;
- Sincronização de tarefas.

## 💻 Tecnologias

- Python 3

## ▶️ Como executar

1. Instale o Python 3 em seu computador.
2. Clone este repositório ou faça o download dos arquivos.
3. Execute o arquivo desejado pelo terminal:

```bash
python nome_do_arquivo.py
```
